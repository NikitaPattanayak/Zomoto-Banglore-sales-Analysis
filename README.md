# Zomoto-Banglore-sales-Analysis
Data analysis project exploring Zomato's restaurant sales and customer trends in Bangalore using Python. Features insights on popular cuisines, pricing, and ratings.

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Analysis Performed
- Data Cleaning
- Null Value Handling
- Visualization
- Correlation Analysis
- Encoding
- Linear Regression

## Machine Learning
Predicted restaurant ratings based on votes.

## Dataset
Kaggle Zomato Bangalore Dataset
